# Hadron Physics Theory Scraper

ハドロン物理学の理論論文を収集・管理するWebアプリケーションです。arXivや学術ジャーナルから最新の論文を自動取得し、キーワード検索やお気に入り管理、メールダイジェスト配信機能を提供します。

## 機能

### 論文の自動収集
- **arXiv統合**: hep-ph、hep-th、hep-lat、nucl-thカテゴリから論文を自動取得
- **学術ジャーナル対応**: 主要な物理学ジャーナルからの論文収集
- **ハドロン物理学フィルタリング**: QCD、クォーク、グルーオン、メソン、バリオンなどのキーワードで自動フィルタリング

### 検索とフィルタリング
- **キーワード検索**: 物理学関連のキーワードで論文を絞り込み
- **フルテキスト検索**: タイトルやアブストラクトから論文を検索
- **ソース別フィルタ**: arXivやジャーナル別に論文を表示

### ユーザー機能
- **ユーザー登録・ログイン**: 安全な認証システム
- **お気に入り管理**: 興味のある論文をブックマーク
- **メールダイジェスト**: 毎日の新着論文をメールで配信（オプション）

### 論文詳細表示
- タイトル、著者、アブストラクト、公開日の表示
- 論文ソースへの直接リンク
- お気に入りへの追加/削除
